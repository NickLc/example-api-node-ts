# APP API

Servicio API REST como ejemplo.

Se simula una base de datos usando un archivo JSON.

## Instalación

Instalar dependencias

```bash
$ npm install
```

## Ejecución

Ejecutar en modo desarrollo

```bash
$ npm run dev
```

## Probar API

Abrir en el navegador la siguiente URL

```bash
http://localhost:8080/api/v1/docs
```
